# Files

## VSCode関連
### .vscode/
VisualStudioCodeの設定等
ローカル環境のことなのでソース管理対象外

## Git関連
### .git/
Gitリポジトリの 内部管理データを保存
### .gitignore
Gitで 追跡しないファイルを指定する設定ファイル
### .github/workflows/build.yml
GitHub Actions の CI/CDワークフロー定義ファイル

## Maven関連
### mvnw.cmd
Maven Wrapper の Windows用起動スクリプト
### mvnw
Maven Wrapper の Unix/Linux/macOS用起動スクリプト
### .mvn/wrapper/maven-wrapper.properties
Maven Wrapper の 設定ファイル、Maven バージョンを指定
### pom.xml
ビルド設定ファイル

## Docker関連
### Dockerfile
Dockerイメージを作成時のビルド手順を記述
### .dockerignore
ビルドコンテキストへ送らないファイルを指定する
